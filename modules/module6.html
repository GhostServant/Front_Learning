Циклы: for, while, do while

Циклы позволяют повторять выполнение блока кода до тех пор, пока условие истинно. В JavaScript существуют три основных типа циклов: for, while и do while.

Задачи для практики:

  • Написать цикл for, который выводит числа от 1 до 10 в консоль.
  • Использовать цикл while для выполнения операции, пока условие истинно.
  • Написать цикл do while, который выполняет операцию хотя бы один раз, даже если условие ложно.

Нюансы:

  • Циклы позволяют повторять выполнение блока кода до тех пор, пока условие истинно.
  • Важно понимать разницу между for, while и do while и выбирать подходящий цикл для конкретной задачи.

Пример использования цикла for:


for (let i = 1; i <= 10; i++) {
    console.log(i);
}

Пример использования цикла while:


let i = 1;
while (i <= 10) {
    console.log(i);
    i++;
}

Пример использования цикла do while:


let i = 1;
do {
    console.log(i);
    i++;
} while (i <= 10);

LocalStorage, sessionStorage

LocalStorage и sessionStorage позволяют хранить данные на стороне клиента. LocalStorage сохраняет данные даже после закрытия браузера, тогда как sessionStorage очищается после закрытия вкладки или окна браузера.

Задачи для практики:

  • Сохранить данные в localStorage и затем извлечь их.
  • Использовать sessionStorage для хранения данных, которые должны быть доступны только в течение сессии.

Нюансы:

  • LocalStorage и sessionStorage позволяют хранить данные на стороне клиента.
  • LocalStorage сохраняет данные даже после закрытия браузера, тогда как sessionStorage очищается после закрытия вкладки или окна браузера.

Пример использования localStorage:


// Сохранение данных в localStorage
localStorage.setItem('name', 'John');

// Извлечение данных из localStorage
let name = localStorage.getItem('name');
console.log(name); // Вывод: John

Пример использования sessionStorage:


// Сохранение данных в sessionStorage
sessionStorage.setItem('age', 30);

// Извлечение данных из sessionStorage
let age = sessionStorage.getItem('age');
console.log(age); // Вывод: 30

Todo list, alert, confirm, prompt

Функции alert, confirm и prompt позволяют взаимодействовать с пользователем, отображая модальные окна с сообщениями, подтверждениями и запросами ввода данных.

Задачи для практики:

  • Создать простое приложение Todo list, которое позволяет добавлять, удалять и отмечать задачи как выполненные.
  • Использовать alert для отображения сообщений, confirm для подтверждения действий и prompt для ввода данных от пользователя.

Нюансы:

  • Функции alert, confirm и prompt являются встроенными функциями для взаимодействия с пользователем.
  • Важно понимать, как эти функции работают и как они могут быть использованы для улучшения пользовательского опыта.

Пример использования alert, confirm и prompt:


// Пример использования alert
alert('Привет, мир!');

// Пример использования confirm
let result = confirm('Вы уверены?');
console.log(result); // Вывод: true или false

// Пример использования prompt
let userInput = prompt('Введите ваше имя:');
console.log(userInput); // Вывод: введенное имя

Области видимости

Области видимости определяют, где переменные и функции доступны в коде. В JavaScript существуют глобальная, функциональная и блочная области видимости.

Задачи для практики:

  • Написать функцию, которая демонстрирует разницу между глобальной и локальной областями видимости.
  • Использовать блочную область видимости с let и const.

Нюансы:

  • Области видимости определяют, где переменные и функции доступны в коде.
  • Важно понимать разницу между глобальной, функциональной и блочной областями видимости.

Пример демонстрации разницы между глобальной и локальной областями видимости:


let globalVar = 'Я глобальная переменная';

function exampleFunction() {
    let localVar = 'Я локальная переменная';
    console.log(localVar); // Вывод: Я локальная переменная
}

exampleFunction();
console.log(globalVar); // Вывод: Я глобальная переменная

Пример использования блочной области видимости с let и const:


if (true) {
    let blockVar = 'Я блочная переменная';
    const blockConst = 'Я блочная константа';
    console.log(blockVar); // Вывод: Я блочная переменная
    console.log(blockConst); // Вывод: Я блочная константа
}

// Ошибка: blockVar и blockConst недоступны за пределами блока
// console.log(blockVar);
// console.log(blockConst);

Замыкания

Замыкания позволяют функции запоминать и иметь доступ к переменным из своей лексической области видимости. Это означает, что функция может использовать переменные, объявленные вне её, даже после завершения выполнения внешней функции.

Задачи для практики:

  • Создать функцию, которая возвращает другую функцию, использующую замыкание.
  • Использовать замыкание для сохранения состояния между вызовами функции.

Нюансы:

  • Замыкания позволяют функции запоминать и иметь доступ к переменным из своей лексической области видимости.
  • Важно понимать, как замыкания работают и как они могут быть использованы для создания модулей и сохранения состояния.

Пример функции, возвращающей другую функцию с использованием замыкания:


function createCounter() {
    let count = 0;
    return function() {
        count++;
        return count;
    };
}

const counter = createCounter();
console.log(counter()); // Вывод: 1
console.log(counter()); // Вывод: 2

Пример использования замыкания для сохранения состояния между вызовами функции:


function createGreeting(name) {
    return function(greeting) {
        console.log(`${greeting}, ${name}!`);
    };
}

const greetJohn = createGreeting('John');
greetJohn('Hello'); // Вывод: Hello, John!
greetJohn('Hi'); // Вывод: Hi, John!

DOM, window, document, this

Объекты window и document предоставляют доступ к окну браузера и DOM соответственно. Ключевое слово this указывает на текущий контекст выполнения и может изменяться в зависимости от контекста.

Задачи для практики:

  • Использовать объект window для выполнения операций, связанных с окном браузера.
  • Манипулировать элементами DOM с помощью объекта document.
  • Понять, как ключевое слово this работает в различных контекстах.

Нюансы:

  • Объекты window и document предоставляют доступ к окну браузера и DOM соответственно.
  • Ключевое слово this указывает на текущий контекст выполнения и может изменяться в зависимости от контекста.

Пример использования объекта window:


// Открыть новое окно
window.open('https://www.example.com', '_blank');

// Закрыть текущее окно
window.close();

// Переместить окно
window.moveTo(100, 100);

Пример манипуляции элементами DOM с помощью объекта document:


// Создать новый элемент
let newElement = document.createElement('div');

// Добавить текст в элемент
newElement.textContent = 'Привет, мир!';

// Добавить элемент в DOM
document.body.appendChild(newElement);

// Изменить стиль элемента
newElement.style.color = 'blue';

Пример использования ключевого слова this:


function Person(name) {
    this.name = name;
    this.greet = function() {
        console.log(`Привет, ${this.name}!`);
    };
}

const john = new Person('John');
john.greet(); // Вывод: Привет, John!

Манипуляция DOM

Манипуляция DOM позволяет динамически изменять содержимое и структуру веб-страницы. Это включает создание, изменение и удаление элементов, а также изменение их стилей и атрибутов.

Задачи для практики:

  • Создать, изменить и удалить элементы DOM с помощью методов createElement, appendChild, removeChild.
  • Изменить стили и атрибуты элементов DOM.

Нюансы:

  • Манипуляция DOM позволяет динамически изменять содержимое и структуру веб-страницы.
  • Важно понимать, как работают различные методы и свойства DOM для эффективного управления элементами.

Пример создания, изменения и удаления элементов DOM:


// Создать новый элемент
let newElement = document.createElement('div');

// Добавить текст в элемент
newElement.textContent = 'Привет, мир!';

// Добавить элемент в DOM
document.body.appendChild(newElement);

// Удалить элемент из DOM
document.body.removeChild(newElement);

Пример изменения стилей и атрибутов элементов DOM:


// Получить элемент по ID
let element = document.getElementById('myElement');

// Изменить стиль элемента
element.style.color = 'blue';
element.style.fontSize = '20px';

// Изменить атрибут элемента
element.setAttribute('data-info', 'some value');

// Получить значение атрибута
let attributeValue = element.getAttribute('data-info');
console.log(attributeValue); // Вывод: some value

События DOM и события

События DOM позволяют реагировать на действия пользователя и изменения состояния элементов. Это включает клики, ввод текста, изменение значений и другие события.

Задачи для практики:

  • Добавить обработчики событий для различных элементов DOM, таких как клики, ввод текста и изменение значений.
  • Использовать методы addEventListener и removeEventListener для управления событиями.

Нюансы:

  • События DOM позволяют реагировать на действия пользователя и изменения состояния элементов.
  • Важно понимать, как работают события и как они могут быть использованы для создания интерактивных веб-приложений.

Пример добавления обработчика события для клика:


// Получить элемент по ID
let button = document.getElementById('myButton');

// Добавить обработчик события для клика
button.addEventListener('click', function() {
    alert('Кнопка нажата!');
});

Пример добавления обработчика события для ввода текста:


// Получить элемент по ID
let input = document.getElementById('myInput');

// Добавить обработчик события для ввода текста
input.addEventListener('input', function(event) {
    console.log(`Введено: ${event.target.value}`);
});

Пример удаления обработчика события:


// Получить элемент по ID
let button = document.getElementById('myButton');

// Функция обработчика события
function handleClick() {
    alert('Кнопка нажата!');
}

// Добавить обработчик события для клика
button.addEventListener('click', handleClick);

// Удалить обработчик события для клика
button.removeEventListener('click', handleClick);
