Вводный урок по JS

JavaScript (JS) - это язык программирования, который используется для создания интерактивных веб-страниц. Он позволяет добавлять динамическое поведение на веб-страницы, такие как обновление контента, валидация форм, анимации и многое другое.

Задачи для практики:

  • Написать простой скрипт, который выводит сообщение "Hello, World!" в консоль.
  • Создать переменные разных типов и вывести их значения.

Нюансы:

  • JavaScript является языком программирования с динамической типизацией.
  • Важно понимать разницу между var, let и const.

Пример скрипта, который выводит сообщение "Hello, World!" в консоль:


console.log("Hello, World!");

Стандартная библиотека

Стандартная библиотека JavaScript предоставляет множество встроенных объектов и методов для работы с данными и выполнения различных задач.

Задачи для практики:

  • Использовать встроенные методы для работы с массивами (например, push, pop, map, filter).
  • Работать с объектами и их методами (например, Object.keys, Object.values).

Нюансы:

  • Стандартная библиотека JavaScript предоставляет множество встроенных объектов и методов.
  • Важно изучать документацию, такую как MDN или webreference, для понимания всех возможностей.

Пример использования встроенных методов для работы с массивами:


let numbers = [1, 2, 3, 4, 5];
numbers.push(6); // Добавляет элемент в конец массива
numbers.pop(); // Удаляет последний элемент массива
let doubled = numbers.map(num => num * 2); // Удваивает каждый элемент массива
let evenNumbers = numbers.filter(num => num % 2 === 0); // Фильтрует четные числа

Подключение JS

JavaScript можно подключать к HTML-документу как внешним файлом, так и встроенным кодом.

Задачи для практики:

  • Подключить внешний JavaScript файл к HTML-документу.
  • Вставить JavaScript код непосредственно в HTML-документ с помощью тега <script>.

Нюансы:

  • JavaScript можно подключать как внешним файлом, так и встроенным кодом.
  • Подключение внешнего файла позволяет разделить логику и представление, что улучшает структуру проекта.

Пример подключения внешнего JavaScript файла:


<script src="script.js"></script>

Пример встроенного JavaScript кода:


<script>
console.log("Hello, World!");
</script>

Переменные

Переменные используются для хранения данных, которые могут изменяться в процессе выполнения программы.

Задачи для практики:

  • Объявить переменные с использованием let и const.
  • Изменить значение переменной и вывести его в консоль.

Нюансы:

  • let используется для объявления переменных, значение которых может изменяться.
  • const используется для объявления констант, значение которых не может быть изменено после инициализации.

Пример объявления переменных с использованием let и const:


let name = "John"; // Переменная, значение которой можно изменить
const age = 30; // Константа, значение которой нельзя изменить
name = "Jane"; // Изменение значения переменной
console.log(name); // Вывод значения переменной

Функции

Функции позволяют группировать блоки кода для повторного использования. Они могут принимать аргументы и возвращать значения.

Задачи для практики:

  • Написать функцию, которая принимает два аргумента и возвращает их сумму.
  • Создать функцию, которая выводит сообщение в консоль при вызове.

Нюансы:

  • Функции могут быть объявлены с помощью ключевого слова function или как стрелочные функции.
  • Важно понимать разницу между объявлением функции и функциональным выражением.

Пример функции, которая принимает два аргумента и возвращает их сумму:


function sum(a, b) {
    return a + b;
}
console.log(sum(2, 3)); // Вывод: 5

Пример функции, которая выводит сообщение в консоль при вызове:


function greet() {
    console.log("Hello, World!");
}
greet(); // Вывод: Hello, World!

Логирование и дебаггинг

Логирование и дебаггинг помогают отслеживать выполнение кода и находить ошибки.

Задачи для практики:

  • Использовать console.log для вывода значений переменных и результатов выполнения функций.
  • Использовать debugger для пошагового выполнения кода и анализа ошибок.

Нюансы:

  • Логирование помогает отслеживать выполнение кода и находить ошибки.
  • Встроенные инструменты разработчика в браузере предоставляют мощные средства для дебаггинга.

Пример использования console.log для вывода значений переменных:


let name = "John";
console.log(name); // Вывод: John

Пример использования debugger для пошагового выполнения кода:


function example() {
    let a = 1;
    debugger; // Останавливает выполнение кода на этой строке
    let b = 2;
    console.log(a + b);
}
example();

Модули

Модули позволяют организовать код и повторно использовать его в разных частях приложения.

Задачи для практики:

  • Создать модуль и экспортировать функции или переменные с помощью export.
  • Импортировать модуль в другой файл с помощью import.

Нюансы:

  • Модули позволяют организовать код и повторно использовать его в разных частях приложения.
  • Важно понимать разницу между именованным экспортом и экспортом по умолчанию.

Пример создания модуля и экспорта функции:


// module.js
export function greet() {
    console.log("Hello, World!");
}

Пример импорта модуля в другой файл:


// main.js
import { greet } from './module.js';
greet(); // Вывод: Hello, World!
