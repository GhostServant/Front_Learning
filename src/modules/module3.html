Свойство position

Свойство position в CSS определяет метод позиционирования элемента. Возможные значения:

  • static (по умолчанию): элемент позиционируется в обычном потоке документа.
  • relative: элемент позиционируется относительно своего нормального положения.
  • absolute: элемент позиционируется относительно ближайшего позиционированного предка (не static).
  • fixed: элемент позиционируется относительно окна просмотра.
  • sticky: элемент позиционируется как relative до тех пор, пока не достигнет определенного порога прокрутки, после чего становится fixed.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Position Example</title>
    <style>
        .relative {
            position: relative;
            top: 20px;
            left: 20px;
        }
        .absolute {
            position: absolute;
            top: 50px;
            left: 50px;
        }
        .fixed {
            position: fixed;
            bottom: 0;
            right: 0;
        }
        .sticky {
            position: sticky;
            top: 0;
        }
    </style>
</head>
<body>
    <div class="relative">Relative Position</div>
    <div class="absolute">Absolute Position</div>
    <div class="fixed">Fixed Position</div>
    <div style="height: 1000px;">Scroll down to see sticky position</div>
    <div class="sticky">Sticky Position</div>
</body>
</html>

Псевдоклассы и псевдоэлементы

Псевдоклассы и псевдоэлементы позволяют применять стили к элементам в определенных состояниях или частях элементов.

  • Псевдоклассы: :hover, :focus, :nth-child(), :first-child, :last-child.
  • Псевдоэлементы: ::before, ::after, ::first-line, ::first-letter.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Pseudo-classes and Pseudo-elements Example</title>
    <style>
        .hover-example:hover {
            background-color: yellow;
        }
        .focus-example:focus {
            outline: 2px solid blue;
        }
        .before-example::before {
            content: ""Before: ";
            color: red;
        }
        .after-example::after {
            content: "" - After";
            color: green;
        }
    </style>
</head>
<body>
    <div class="hover-example">Hover over me</div>
    <input type="text" class="focus-example" placeholder="Focus on me">
    <p class="before-example">Text with before pseudo-element</p>
    <p class="after-example">Text with after pseudo-element</p>
</body>
</html>

Медиа выражения

Медиа выражения (media queries) позволяют применять стили в зависимости от характеристик устройства, таких как ширина экрана, ориентация и разрешение.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Media Queries Example</title>
    <style>
        body {
            background-color: lightblue;
        }
        @media (max-width: 600px) {
            body {
                background-color: lightcoral;
            }
        }
        @media (orientation: portrait) {
            body {
                background-color: lightgreen;
            }
        }
    </style>
</head>
<body>
    <p>Resize the browser window to see the media queries in action.</p>
</body>
</html>

Зачетная работа по модулю 3 "Базовый CSS"

Зачетная работа по модулю "Базовый CSS" включает создание веб-страницы с использованием основных CSS-свойств и техник.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Final Project</title>
    <style>
        body {
            font-family: Arial, sans-serif;
            margin: 0;
            padding: 0;
            background-color: #f4f4f4;
        }
        header {
            background-color: #333;
            color: white;
            padding: 10px 0;
            text-align: center;
        }
        nav {
            background-color: #444;
            color: white;
            padding: 10px;
            text-align: center;
        }
        nav a {
            color: white;
            margin: 0 10px;
            text-decoration: none;
        }
        main {
            padding: 20px;
        }
        footer {
            background-color: #333;
            color: white;
            text-align: center;
            padding: 10px 0;
            position: fixed;
            width: 100%;
            bottom: 0;
        }
    </style>
</head>
<body>
    <header>
        <h1>My Website</h1>
    </header>
    <nav>
        <a href="#home">Home</a>
        <a href="#about">About</a>
        <a href="#contact">Contact</a>
    </nav>
    <main>
        <section id="home">
            <h2>Home</h2>
            <p>Welcome to my website!</p>
        </section>
        <section id="about">
            <h2>About</h2>
            <p>This is a sample website for the final project.</p>
        </section>
        <section id="contact">
            <h2>Contact</h2>
            <p>You can contact me at example@example.com</p>
        </section>
    </main>
    <footer>
        <p>© 2023 My Website</p>
    </footer>
</body>
</html>

CSS и способы его подключения

CSS можно подключать к HTML-документу несколькими способами:

  • Встроенные стили: стили применяются непосредственно к элементу с помощью атрибута style.
  • Внутренние стили: стили применяются внутри тега <style> в секции <head>.
  • Внешние стили: стили применяются из отдельного CSS-файла с помощью тега <link>.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>CSS Linking Methods</title>
    <style>
        /* Внутренние стили */
        body {
            font-family: Arial, sans-serif;
        }
        .internal {
            color: blue;
        }
    </style>
    <link rel="stylesheet" href="styles.css"> 
</head>
<body>
    <p class="internal">This is an internal style.</p>
    <p class="external">This is an external style.</p>
    <p style="color: green;">This is an inline style.</p>
</body>
</html>

Основные селекторы, специфичность, каскадность

Селекторы в CSS используются для выбора элементов, к которым будут применяться стили. Основные типы селекторов:

  • Элементные селекторы: выбирают все элементы определенного типа (например, p, div).
  • Классовые селекторы: выбирают элементы с определенным классом (например, .classname).
  • Идентификационные селекторы: выбирают элемент с определенным идентификатором (например, #id).

Специфичность определяет, какой стиль будет применен, если несколько стилей конфликтуют. Каскадность определяет порядок применения стилей.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Selectors, Specificity, and Cascade</title>
    <style>
        /* Элементный селектор */
        p {
            color: blue;
        }
        /* Классовый селектор */
        .class-selector {
            font-weight: bold;
        }
        /* Идентификационный селектор */
        #id-selector {
            color: red;
        }
        /* Специфичность */
        .class-selector.another-class {
            background-color: yellow;
        }
    </style>
</head>
<body>
    <p>This is a paragraph.</p>
    <p class="class-selector">This is a paragraph with a class.</p>
    <p id="id-selector">This is a paragraph with an ID.</p>
    <p class="class-selector another-class">This is a paragraph with multiple classes.</p>
</body>
</html>

Шрифты

Шрифты в CSS настраиваются с помощью свойств font-family, font-size, font-weight, font-style и других. Можно использовать веб-шрифты с помощью @font-face или подключения через Google Fonts.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Fonts Example</title>
    <style>
        @font-face {
            font-family: 'CustomFont';
            src: url('customfont.woff2') format('woff2');
        }
        body {
            font-family: 'Arial', sans-serif;
        }
        .custom-font {
            font-family: 'CustomFont', sans-serif;
            font-size: 20px;
            font-weight: bold;
        }
        .italic-text {
            font-style: italic;
        }
    </style>
</head>
<body>
    <p>This is a regular text.</p>
    <p class="custom-font">This is a text with a custom font.</p>
    <p class="italic-text">This is a text with italic style.</p>
</body>
</html>

Свойство display

Свойство display определяет, как элемент должен быть отображен. Возможные значения:

  • block: элемент занимает всю ширину контейнера.
  • inline: элемент отображается в строке.
  • inline-block: элемент отображается в строке, но может иметь ширину и высоту.
  • none: элемент не отображается.
  • flex: элемент становится flex-контейнером.
  • grid: элемент становится grid-контейнером.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Display Property</title>
    <style>
        .block {
            display: block;
            width: 100%;
            background-color: lightblue;
        }
        .inline {
            display: inline;
            background-color: lightgreen;
        }
        .inline-block {
            display: inline-block;
            width: 200px;
            background-color: lightcoral;
        }
        .none {
            display: none;
        }
    </style>
</head>
<body>
    <div class="block">Block Element</div>
    <span class="inline">Inline Element</span>
    <div class="inline-block">Inline-Block Element</div>
    <div class="none">Hidden Element</div>
</body>
</html>

Display: flex

Свойство display: flex создает flex-контейнер, который позволяет гибко управлять расположением дочерних элементов. Основные свойства flex-контейнера:

  • flex-direction: определяет направление главной оси (row, row-reverse, column, column-reverse).
  • justify-content: выравнивает элементы по главной оси (flex-start, flex-end, center, space-between, space-around).
  • align-items: выравнивает элементы по поперечной оси (flex-start, flex-end, center, baseline, stretch).

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Flexbox Example</title>
    <style>
        .flex-container {
            display: flex;
            flex-direction: row;
            justify-content: space-between;
            align-items: center;
            height: 100px;
            background-color: lightblue;
        }
        .flex-item {
            background-color: lightcoral;
            padding: 10px;
            margin: 5px;
        }
    </style>
</head>
<body>
    <div class="flex-container">
        <div class="flex-item">Item 1</div>
        <div class="flex-item">Item 2</div>
        <div class="flex-item">Item 3</div>
    </div>
</body>
</html>

Display: grid

Свойство display: grid создает grid-контейнер, который позволяет создавать сложные макеты с помощью сетки. Основные свойства grid-контейнера:

  • grid-template-columns: определяет количество и размер столбцов.
  • grid-template-rows: определяет количество и размер строк.
  • grid-gap: определяет расстояние между ячейками сетки.
  • grid-template-areas: определяет именованные области в сетке.

Пример кода:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Grid Example</title>
    <style>
        .grid-container {
            display: grid;
            grid-template-columns: repeat(3, 1fr);
            grid-template-rows: 100px 100px;
            grid-gap: 10px;
            background-color: lightblue;
        }
        .grid-item {
            background-color: lightcoral;
            padding: 20px;
            text-align: center;
        }
    </style>
</head>
<body>
    <div class="grid-container">
        <div class="grid-item">Item 1</div>
        <div class="grid-item">Item 2</div>
        <div class="grid-item">Item 3</div>
        <div class="grid-item">Item 4</div>
        <div class="grid-item">Item 5</div>
        <div class="grid-item">Item 6</div>
    </div>
</body>
</html>
