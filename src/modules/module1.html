Основные теги

HTML (HyperText Markup Language) - это стандартный язык разметки для создания веб-страниц. Он используется для структурирования контента на вебе.

Основные теги HTML включают:

  • <html> - корневой элемент HTML-документа.
  • <head> - содержит метаданные о документе, такие как заголовок и ссылки на стили.
  • <body> - основной контент веб-страницы, который видит пользователь.
  • <meta> - предоставляет метаданные, такие как кодировка и описание страницы.
  • <title> - задает заголовок страницы, отображаемый в вкладке браузера.

Пример структуры HTML-документа:


<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Заголовок страницы</title>
</head>
<body>
    <h1>Добро пожаловать на мой сайт</h1>
</body>
</html>

Iframe

Тег <iframe> используется для встраивания другого документа в текущую страницу. Это позволяет отображать контент с других сайтов или страниц.

Пример использования <iframe> для встраивания видео:

Пример встраивания видео с YouTube:


<iframe width="100%"
height="315"
src="https://www.youtube.com/embed/uCNWuSbPnt4"
title="YouTube video player"
frameborder="0"
allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture"
allowfullscreen>
</iframe>

Семантика

Семантические теги помогают улучшить структуру документа и его доступность для пользователей и поисковых систем. Они делают код более понятным и логичным.

Использование семантических тегов также способствует SEO (поисковой оптимизации), так как поисковые системы могут легче интерпретировать содержание страницы.

  • <h1> - <h6> - заголовки различного уровня, от самого важного (<h1>) до менее важного (<h6>).
  • <section> - определяет раздел документа.
  • <main> - основной контент страницы.
  • <header> - заголовок страницы или раздела.
  • <footer> - нижний колонтитул страницы или раздела.

Пример семантической структуры:


<header>
    <h1>Заголовок страницы</h1>
</header>
<main>
    <section>
        <h2>Подзаголовок раздела</h2>
        <p>Некоторый текст в этом разделе.</p>
    </section>
</main>
<footer>
    <p>© 2024 Мой сайт</p>
</footer>

Таблицы

Таблицы используются для представления данных в табличном формате. Они состоят из строк и столбцов, что позволяет организовать информацию в удобочитаемом виде.

Таблицы могут быть полезны для отображения статистики, расписаний, сравнений и других данных, которые требуют структурированного представления.

Пример таблицы с данными о пользователях:

Пример таблицы:


<table>
    <thead>
        <tr>
            <th>Имя</th>
            <th>Возраст</th>
            <th>Город</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Иван</td>
            <td>25</td>
            <td>Москва</td>
        </tr>
        <tr>
            <td>Мария</td>
            <td>30</td>
            <td>Санкт-Петербург</td>
        </tr>
    </tbody>
</table>
Имя Возраст Город
Иван 25 Москва
Мария 30 Санкт-Петербург

Формы

Формы позволяют пользователям вводить данные и отправлять их на сервер. Они являются важной частью взаимодействия с пользователем на веб-сайтах.

Формы могут включать различные элементы, такие как текстовые поля, кнопки, флажки и выпадающие списки, что делает их гибкими для различных сценариев.

Пример формы для ввода данных:

Пример формы:


<form action="/submit" method="post">
    <label for="name">Имя:</label>
    <input type="text" id="name" name="name" required>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>
    <input type="submit" value="Отправить">
</form>
