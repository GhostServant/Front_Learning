Операторы сравнения

Операторы сравнения используются для сравнения значений и возвращают булево значение (true или false). В JavaScript существует два типа сравнений: строгое и нестрогое.

Задачи для практики:

  • Написать скрипт, который сравнивает два числа и выводит результат сравнения в консоль.
  • Использовать операторы ==, ===, !=, !==, >, <, >=, <= для различных типов данных.

Нюансы:

  • Операторы == и != выполняют нестрогое сравнение, тогда как === и !== выполняют строгое сравнение.
  • Важно понимать, как JavaScript преобразует типы данных при сравнении. Например, при использовании ==, JavaScript может преобразовать строку в число для сравнения.

Пример использования операторов сравнения:


let a = 5;
let b = '5';
console.log(a == b);  // true, нестрогое сравнение
console.log(a === b); // false, строгое сравнение
console.log(a != b);  // false, нестрогое сравнение
console.log(a !== b); // true, строгое сравнение
console.log(a > b);  // false, сравнение
console.log(a < b);  // false, сравнение
console.log(a >= b); // true, сравнение
console.log(a <= b); // true, сравнение

Тернарные операторы

Тернарный оператор является сокращенной формой условного оператора if-else. Он позволяет выполнить проверку условия и вернуть одно из двух значений в зависимости от результата проверки.

Задачи для практики:

  • Использовать тернарный оператор для проверки условия и вывода соответствующего сообщения.
  • Переписать условный оператор if-else с использованием тернарного оператора.

Нюансы:

  • Тернарный оператор является сокращенной формой условного оператора if-else.
  • Важно использовать тернарный оператор для простых условий, чтобы код оставался читаемым. Для сложных условий лучше использовать полную форму if-else.

Пример использования тернарного оператора:


let age = 18;
let message = age >= 18 ? 'Вы совершеннолетний' : 'Вы несовершеннолетний';
console.log(message); // Вывод: Вы совершеннолетний

Пример переписанного условного оператора if-else с использованием тернарного оператора:


let score = 85;
let grade = score >= 90 ? 'A' :
               score >= 80 ? 'B' :
               score >= 70 ? 'C' :
               score >= 60 ? 'D' : 'F';
console.log(`Ваша оценка: ${grade}`); // Вывод: Ваша оценка: B

Задачи для практики:

  • Использовать методы для работы с различными типами данных (например, методы строк и массивов).

Нюансы:

  • JavaScript поддерживает несколько типов данных, включая примитивные и объектные типы.
  • Важно понимать, как работать с каждым типом данных и какие методы доступны для них.

Пример создания переменных различных типов данных:


let number = 42;

let array = [1, 2, 3, 4, 5];

let str = 'Hello, World!';

let arr = [1, 2, 3, 4, 5];

Базовые операторы

Базовые операторы являются основой для выполнения различных операций в JavaScript. Они включают арифметические операторы, операторы присваивания, операторы сравнения, логические операторы и операторы строковой конкатенации.

Задачи для практики:

  • Использовать арифметические операторы (+, -, *, /, %) для выполнения математических операций.
  • Применить операторы присваивания (=, +=, -=, *=, /=, %=) для изменения значений переменных.
  • Использовать операторы сравнения (==, ===, !=, !==, >, <, >=, <=) для сравнения значений.
  • Использовать логические операторы (&&, ||, !) для выполнения логических операций.

Нюансы:

  • Базовые операторы являются основой для выполнения различных операций в JavaScript.
  • Важно понимать порядок выполнения операторов и использовать скобки для явного указания приоритета.

Пример использования базовых операторов:


let x = 10;
let y = 5;
console.log(x + y); // 15
console.log(x - y); // 5
console.log(x * y); // 50
console.log(x / y); // 2
console.log(x % y); // 0

x += y; // x = x + y
console.log(x); // 15

let a = 5;
let b = '5';
console.log(a == b);  // true, нестрогое сравнение
console.log(a === b); // false, строгое сравнение

let c = true;
let d = false;
console.log(c && d); // false
console.log(c || d); // true
console.log(!c); // false

Условные операторы

Условные операторы позволяют выполнять различные блоки кода в зависимости от условий. В JavaScript существуют операторы if, else if, else и switch.

Задачи для практики:

  • Написать скрипт, который использует условные операторы if, else if, else для проверки различных условий.
  • Использовать оператор switch для выполнения различных действий в зависимости от значения переменной.

Нюансы:

  • Условные операторы позволяют выполнять различные блоки кода в зависимости от условий.
  • Важно использовать условные операторы для управления потоком выполнения программы.

Пример использования условных операторов:


let age = 18;
if (age >= 18) {
    console.log('Вы совершеннолетний');
} else {
    console.log('Вы несовершеннолетний');
}

let day = 3;
switch (day) {
    case 1:
        console.log('Понедельник');
        break;
    case 2:
        console.log('Вторник');
        break;
    case 3:
        console.log('Среда');
        break;
    default:
        console.log('Другой день');
}

Неявное и явное преобразование

Преобразование типов данных может быть неявным и явным. Неявное преобразование происходит автоматически в определенных контекстах, тогда как явное преобразование выполняется с использованием специальных функций.

Задачи для практики:

  • Привести примеры неявного преобразования типов данных и объяснить, как это происходит.
  • Использовать явное преобразование типов данных с помощью функций Number(), String(), Boolean().

Нюансы:

  • Неявное преобразование типов данных происходит автоматически в определенных контекстах.
  • Явное преобразование типов данных позволяет контролировать процесс преобразования и избежать неожиданных результатов.

Пример неявного преобразования типов данных:


let num = 5;
let str = '10';
console.log(num + str); // '510', строка

Пример явного преобразования типов данных:


let num = '10';
let number = Number(num);
console.log(number); // 10, число

let bool = 1;
let boolean = Boolean(bool);
console.log(boolean); // true, булево значение

let str = 123;
let string = String(str);
console.log(string); // '123', строка

Массивы и его методы, 1 часть

Массивы являются важной структурой данных в JavaScript. Они позволяют хранить список значений и предоставляют множество методов для работы с этими значениями.

Задачи для практики:

  • Создать массив и выполнить базовые операции, такие как добавление, удаление и изменение элементов.
  • Использовать методы push, pop, shift, unshift, slice, splice для работы с массивами.

Нюансы:

  • Массивы являются важной структурой данных в JavaScript.
  • Важно понимать, как работают различные методы массивов и как они изменяют исходный массив.

Пример использования методов массивов:


let arr = [1, 2, 3, 4, 5];
arr.push(6); // [1, 2, 3, 4, 5, 6]
arr.pop(); // [1, 2, 3, 4, 5]
arr.shift(); // [2, 3, 4, 5]
arr.unshift(0); // [0, 2, 3, 4, 5]
let sliced = arr.slice(1, 3); // [2, 3]
arr.splice(2, 1, 'a', 'b'); // [0, 2, 'a', 'b', 4, 5]
console.log(arr, sliced);

Массивы и его методы, 2 часть

Методы map, filter, reduce, forEach позволяют выполнять сложные операции с массивами. Эти методы позволяют обрабатывать каждый элемент массива и возвращать новый массив или значение на основе определенных условий.

Задачи для практики:

  • Использовать методы map, filter, reduce, forEach для работы с массивами.
  • Написать функции, которые принимают массив в качестве аргумента и возвращают новый массив на основе определенных условий.

Нюансы:

  • Методы map, filter, reduce, forEach позволяют выполнять сложные операции с массивами.
  • Важно понимать, как эти методы работают и как они могут быть использованы для решения различных задач.

Пример использования методов массивов:


let arr = [1, 2, 3, 4, 5];

// Метод map создает новый массив с результатом вызова указанной функции для каждого элемента массива
let doubled = arr.map(x => x * 2); // [2, 4, 6, 8, 10]
console.log(doubled);

// Метод filter создает новый массив со всеми элементами, прошедшими проверку, implement
let evens = arr.filter(x => x % 2 === 0); // [2, 4]
console.log(evens);

// Метод reduce сводит массив к одному значению, выполняя указанную функцию для каждого элемента массива
let sum = arr.reduce((acc, x) => acc + x, 0); // 15
console.log(sum);

// Метод forEach выполняет указанную функцию для каждого элемента массива
arr.forEach(x => console.log(x)); // 1 2 3 4 5

Объекты

Объекты позволяют хранить и организовывать данные в виде пар "ключ-значение". Объекты в JavaScript являются динамическими, что означает, что вы можете добавлять, изменять и удалять свойства после создания объекта.

Задачи для практики:

  • Создать объект и добавить, изменить, удалить его свойства.
  • Использовать методы объектов, такие как Object.keys, Object.values, Object.entries.

Нюансы:

  • Объекты позволяют хранить и организовывать данные в виде пар "ключ-значение".
  • Важно понимать, как работать с объектами и их методами для эффективного управления данными.

Пример работы с объектами:


// Создание объекта
let person = {
    name: 'John',
    age: 30
};

// Добавление свойства
person.city = 'New York';

// Удаление свойства
delete person.age;

// Изменение свойства
person.name = 'Jane';

// Получение ключей объекта
let keys = Object.keys(person);
console.log(keys); // ['name', 'city']

// Получение значений объекта
let values = Object.values(person);
console.log(values); // ['Jane', 'New York']

// Получение пар ключ-значение объекта
let entries = Object.entries(person);
console.log(entries); // [['name', 'Jane'], ['city', 'New York']]
